# Bananas-Bananas-Bananas

## Overview 
In a group of 4, we determined the mass of potassium in unripe and ripe bananas and plantains via titration and atomic absorption spectroscopy (AAS).

## 🫡 My Contributions
- Processed potassium via dry ashing ripe + unripe banana and plantain flesh
- Performed back titrations to determine potassium concentration in the samples
